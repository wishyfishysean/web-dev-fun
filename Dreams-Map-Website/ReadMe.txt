# 🌌 Dream Archive Mini-Site

A self-contained **HTML/CSS/JS** mini-site that lets visitors explore and contribute to fading dream-worlds.  
The site blends **mood layers**, **visitor input**, and **time-based fading** to create an interactive, ever-changing archive.

---

## ✨ Features
- 🎭 **Mood Layers**
Toggle between *Hopeful*, *Melancholic*, and *Chaotic* aesthetics.  
- 🌍 **Visitor Input**
Users can add their own dream-places, which appear in the archive.  
- ⏳ **Fading Worlds**
Entries older than **7 days** slowly vanish, echoing the fragility of memory.  
- 📝 **Readable Code**
Simple, commented HTML/CSS/JS for easy learning and modification.
